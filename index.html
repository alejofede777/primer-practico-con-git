Trabajo practico en Git

En esta clase vamos a estar aplicando los conocimientos de Git

Lista de comandos vistos en la clase

  • git init: inicializar el repositorio
  • git status: ver el estado del repositorio
  • git add: agregar archivos al repositorio
  • git commit: confirmar los cambios
  • git log: ver el historial de los commits
  • git checkout -b: crear una rama nueva
  • git checkout: cambiar de rama
  • git merge: fusionar rama
